# Undertale Engine Platinum Edition

## About
**Undertale Engine Platinum Edition** Is A Modified Version Of Undertale Engine By TML
That Have More Features That Original Engine That Dont Have 
This Project Is Based By Gamemaker Studio 2 Undertale Engine TML Somehow This Engine Is Almost Easy To Use.


## Featues
*   Battle Turn Flowing (`battle_turn_sans`).
*   Battle Bullet Attacks (`battle_bullet`, `bullet_bone`, `battle_bullet_wbonestab`, `battle_bullet_gb`)
*   Sans Item Example (`Item_Custom`).
*   Debug Mode And Easy Mode (`battle_ui`).
*   Better Enemy Example Fight (`battle_enemy_sans`)
*   Settings Menu (`room_settings`)
*   Border Support (`border`)
*   Multiple Soul Support (`battle_soul_red, blue, orange, etc`)
*   Battle Box Adjustable (`sb`)
*   Platform Green And Purple (`battle_plate, battle_plate_bounce`)
*   Gameover (`room_gameover`)
*   Changeble Head Body Sweat Enemy (`sanshead, sansbody, sansheadsweat`)
*   Sans KR (`hurter`)

## Requirement
*  Gamemaker Version v2024.14
*  IDE v2024.14.0.207
*  Runtimes v2024.14.0.251
